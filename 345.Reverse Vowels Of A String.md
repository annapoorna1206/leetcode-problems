# Reverse Vowels of a String (LeetCode 345) https://leetcode.com/problems/reverse-vowels-of-a-string/

## Problem Statement

Given a string `s`, reverse only the vowels in the string and return the resulting string.

Vowels include:

```text
a, e, i, o, u, A, E, I, O, U
```

### Example

**Input:**
```text
s = "hello"
```

**Output:**
```text
"holle"
```

**Explanation:**
```text
The vowels 'e' and 'o' are swapped.
```

---

## Approach

- Convert the string into a list since strings are immutable.
- Use two pointers:
  - `left` starting from the beginning.
  - `right` starting from the end.
- Move `left` until it points to a vowel.
- Move `right` until it points to a vowel.
- Swap the vowels and move both pointers inward.
- Convert the list back to a string and return it.

---

## Code

```python
class Solution:
    def reverseVowels(self, s: str) -> str:
        s = list(s)
        vow = "aeiouAEIOU"

        left = 0
        right = len(s) - 1

        while left < right:
            while left < right and s[left] not in vow:
                left += 1

            while left < right and s[right] not in vow:
                right -= 1

            s[left], s[right] = s[right], s[left]

            left += 1
            right -= 1

        return "".join(s)
```

## Complexity

- **Time Complexity:** O(n)
- **Space Complexity:** O(n)
